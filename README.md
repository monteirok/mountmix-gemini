# Mountain Mixology

A premium craft cocktail catering landing page built with Next.js 15+, Tailwind CSS v4, and Framer Motion.

## Features

- **Liquid Glass UI**: Custom glassmorphism effects using Tailwind v4.
- **Smooth Animations**: Powered by Framer Motion and Lenis for smooth scrolling.
- **Responsive Design**: Fully mobile-optimized.
- **Interactive Components**:
  - Hover-reveal cocktail ingredients.
  - Glassmorphic service cards.
  - Modal booking form.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Lenis (Smooth Scroll)
- Lucide React (Icons)
